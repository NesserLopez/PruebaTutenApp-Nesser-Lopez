La aplicación móvil tiene por nombre "appPruebaTuten"

Ahí se encuentra solucionado el problema 4 (parte 1 y 2) los componentes
utilizados fueron, Ionic v4 y angular v.7.0.0

para poder replicar la prueba hay que descargar el proyecto, desde
CMD dirigirse a la ruta del proyecto y una vez se encuentre en el 
directorio raiz (appPruebaTuten) ejecutar el comando "ionic serve"
ó simplemente instalar el archivo "prueba tuten app.apk" en un dispositivo
android superior a la version 6.0.0
